# qtls.sh

Simple script en Bash que consulta los protocolos SSL/TLS y cifrados, permitidos en un host remoto

Instalacion.:

git clone https://github.com/hackingyseguridad/qtls/

cd qtls

chmod 777 qtls.sh

Uso.:

sh qtls.sh IP:443
